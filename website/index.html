Why Emu stands out

Features you'll love

Split screen mode

Run two terminals side by side and drop tabs into either pane without breaking flow.

Unlimited tabs

Open as many named sessions as you need. Each tab keeps its own PTY, history, and timestamps.

Jump-to-command

Fly through scrollback one command at a time with prompt-by-prompt navigation that keeps you organized.

Clickable links & filepaths

Cmd-click URLs and domains, or open absolute and relative paths directly in Finder.

GPU rendering

WebGL-accelerated text rendering keeps every character razor sharp and scrolling silky smooth, even in dense output.

Custom color themes

Six curated themes, from Catppuccin to Monokai. Switch instantly to match your style.

Premium typography

JetBrains Mono tuned with pixel-perfect sizing, integer font size, and a clean 1.0 line height for crisp terminal text.

Frosted glass UI

Vibrancy, blur, and brightness work together on the shell, sidebar, and titlebar for a polished native feel.

Output Copy Button

Every entry in the Command Log includes a clipboard icon that copies the full output in one click, clean and shell-free.

Font Zoom

⌘+ and ⌘− scale the terminal font instantly, and ⌘0 snaps back to default while the layout auto-fits the viewport.

Auto Scroll

When you're not at the bottom, a button appears in the lower-right corner so you can jump straight back down.

Highlight-to-Delete

Select any text and press backspace to delete exactly that many characters, making surgical edits feel effortless.
